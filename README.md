# Sola AI - منصة توليد البرومبتات

🚀 **منصة احترافية وفاخرة لتوليد برومبتات عالية الجودة**

## الميزات

✨ **توليد برومبتات متقدمة** - 5 مستويات من الجودة (بسيط → نخبة)
🌍 **دعم ثنائي اللغة** - العربية والإنجليزية
🎨 **تصميم فاخر** - واجهة حديثة مع Glassmorphism
🌓 **موضع داكن/فاتح** - تبديل سلس بين الأوضاع
📥 **تحميل متعدد** - نسخ، PDF، TXT
⚡ **سرعة عالية** - أداء محسّن

## البدء السريع

### المتطلبات
- Node.js 16+
- npm أو yarn

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/T7Tjjjjjjao/sola-ai-.git
cd sola-ai-

# تثبيت الحزم
npm install

# بدء خادم التطوير
npm run dev
```

 الموقع سيكون متاحاً على: http://localhost:3000

## الاستخدام

1. اكتب فكرتك في مربع الإدخال
2. انقر على "توليد البرومبتات"
3. ستحصل على 5 برومبتات بمستويات مختلفة
4. انسخ أو حمّل البرومبتات بصيغة PDF/TXT

## مستويات البرومبتات

1. **بسيط** - برومبت واضح وسهل
2. **متقدم** - مع تفاصيل أكثر
3. **احترافي** - شامل وقوي
4. **خبير** - مع جميع العناصر المتقدمة
5. **نخبة** - الأفضل والأكمل

## التكنولوجيا المستخدمة

- **Next.js 14** - إطار العمل
- **React 18** - مكتبة الواجهة
- **TypeScript** - لغة البرمجة
- **Tailwind CSS** - تصميم الواجهة
- **Framer Motion** - الحركات
- **Zustand** - إدارة الحالة
- **jsPDF** - تحميل PDF
